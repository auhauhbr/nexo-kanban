# Capturas do Nexo

Use resolução aproximada de `1440x900`, esconda dados pessoais e preencha o
quadro com conteúdo que demonstre as funcionalidades.

Salve as capturas nesta pasta com estes nomes:

1. `preview-quadro.png`
   - Visão geral do quadro com pelo menos quatro listas.
   - Mostrar etiquetas, prazos, capas, checklists e uma lista acima do WIP limit.
2. `preview-cartao.png`
   - Modal de cartão aberto.
   - Mostrar descrição, checklist, comentário, histórico e anexo por link.
3. `preview-arquivados.png`
   - Central de Arquivados aberta com cartões e listas disponíveis para restaurar.
4. `preview-autenticacao.png`
   - Tela de cadastro mostrando confirmação e requisitos de senha.

Antes de tirar cada captura:

- Evite nomes, e-mails ou textos pessoais.
- Use dados de demonstração curtos e fáceis de ler.
- Feche ferramentas do navegador e notificações externas.
- Confira se nenhuma senha, token ou URL privada aparece.
